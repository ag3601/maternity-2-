○特殊なコマンドの解説

・座らせる
　その名の通り座らせます。

・情報を見る
　ユーザ名や娘に自分がなんと呼ばれているかを確認できます。
　また、マカナが娘が今どこにいるかを確認する事が出来ます。どちらかがいない場合はこちらから確認してみてください。

・場所を移動する
　最初は行けるところがありませんが、娘が出来てある程度自立し始めると
　（条件を満たしていた場合）マカナの部屋や娘の部屋に行けるようになります。

・ユーザ行動
　ユーザから様々なアクションを起こします。まだまだ機能整備中です。

　　・話しかける
　　　マカナに話しかけます。

　　・ズボンをおろす
　　　自分の股間を露出させます。
　　　ちゃんと時と場合を考えてズボンを下ろしましょう。

　　・えっちなことさせて
　　　えっちなことをさせてくれないか頼んでみます。
　　　仲が良ければ良いほどさせてくれる…かもしれません。

　　・服装リクエスト
　　　少し仲が良ければ選択できるメニューです。
　　　着てほしい服をリクエストする事ができます。

　　・外出する
　　　外に出かけます。現在は服屋でマカナの服を買ってあげることしかできません。
　　　1日に1回しか実行する事ができません。


※ここから下はえっちな事をしている時に選択できるコマンドです

・体を起させるor押し倒す
　押し倒している場合は体を起させます。それ以外の場合は押し倒します。

・服を脱がせる
　服を脱がせます。嫌がられている場合は無理やり脱がします。
　服を着ている場合は下着姿に、下着姿の場合は全裸になります。
　当然ですがパンツを脱がせないと挿入できないので、初期の服装以外ではこのコマンドを実行してからことに及ぶことになります。



○各種設定でいじれる項目の解説

・喋り頻度
　喋り頻度を変更します。

・触り反応の設定
　触り反応を切り替えたり、危なげな触り反応をＯＮ/ＯＦＦできます。

　　・乳首ホイールころがし反応
　　　乳首ホイールころがし反応を舐める・引っ張るで切り替える事が出来ます。

　　・お腹殴り反応
　　　ＯＮにすることで、お腹つつき反応（お腹殴り反応）を開放します。デフォルトではＯＦＦになっています。

　　・性的な行動をフィルタ
　　　ＯＮにすることで、口ホイールころがし反応（フェラ）等のコマンドをえっち中でない限り無効化します。

　　・服脱がすのにホイールを使用
　　　ＯＮにすることで、マウスジェスチャだけでなくホイールころがしで服を脱がせる事が出来るようになります。

・性的な事に関する設定
　主にえっちな事にかかわる事の設定を変えられます。

　　・コンドームの装着
　　　挿入時にコンドームを装着するかどうかを設定できます。

　　・発情期イベントの発生
　　　発情期イベントの発生を許可するか設定できます。

　　・自発的に服を脱ぐ
　　　ムードが高まっている状態で服を着ている場合、服を自発的に脱ぐ事を許可するか設定できます。

　　・触り反応の自動反復
　　　ＯＮにすることで、前回行った触り反応を指定秒ごとに実行するようになります。
